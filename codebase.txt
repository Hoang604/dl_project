## Startup Flows

### Few-Shot Open-Set Evaluation

1. Load pretrained encoder model.
2. Initialize `NearestClassMean` (openNCM) classifier.
3. Sample support set (K-shot) from GSC dataset.
4. Fit classifier prototypes (including random unknown samples).
5. Run inference on test set and compute metrics (AUROC, Accuracy @ 5% FAR).

**Generated:** 2026-05-18
**Last Updated:** 2026-05-18
**Last Verified:** 2026-05-18

## Purpose

Handles the rejection of unknown speech utterances in a few-shot scenario. It uses a prototype-based approach where an "unknown" class is represented by random samples from the target domain.

Evidence: 2306.02161v1.pdf, KWSFSL/classifiers/NCM.py:12

## Key Files

| File | Responsibility | Evidence |
| ---- | -------------- | -------- |
| `KWSFSL/classifiers/NCM.py` | Nearest Class Mean classifier implementation. | KWSFSL/classifiers/NCM.py:12 |
| `KWSFSL/test_fewshots_classifiers_openset.py` | Orchestrates open-set evaluation. | KWSFSL/test_fewshots_classifiers_openset.py:1 |

## Important Flows

### Few-Shot Customization

1. User provides $K$ samples for a new keyword.
2. Encoder generates embeddings for these samples.
3. Prototype is computed as the mean of embeddings.
4. Classifier stores prototypes for inference.

## Purpose

Implements the episodic training protocol to learn a robust embedding space for keyword spotting. It uses Triplet Loss to cluster similar keywords and separate different ones.

Evidence: KWSFSL/metric_learning.py:1, KWSFSL/models/losses/triplet.py:11

## Key Files

| File | Responsibility | Evidence |
| ---- | -------------- | -------- |
| `KWSFSL/metric_learning.py` | Main training loop for episodic learning. | KWSFSL/metric_learning.py:1 |
| `KWSFSL/models/losses/triplet.py` | Implementation of online triplet loss. | KWSFSL/models/losses/triplet.py:11 |

## Important Flows

### Episodic Training

1. Sample $M$ classes and $(S+Q)$ samples per class.
2. Compute embeddings for all samples.
3. Select triplets (anchor, positive, negative).
4. Minimize Triplet Loss to optimize encoder weights.

# Patterns And Conventions

**Generated:** 2026-05-18
**Last Updated:** 2026-05-18
**Last Verified:** 2026-05-18

## Verified Patterns

### Model Registration

- Description: Models are registered using a decorator and retrieved via a central factory function.
- Why it appears to exist: Decouples model definition from training/testing scripts, allowing easy configuration via command-line arguments.
- Examples: `KWSFSL/models/repr_model.py`, `KWSFSL/models/utils.py`
- Evidence: KWSFSL/models/utils.py:5, KWSFSL/models/repr_model.py:144

### Episodic Data Loading

- Description: Data is sampled in "episodes" (N-way, K-shot) rather than standard IID batches.
- Why it appears to exist: Required for Few-Shot Learning to simulate the customization scenario during training.
- Examples: `KWSFSL/data/GSCSpeechData.py`, `KWSFSL/data/MSWCData.py`
- Evidence: KWSFSL/data/GSCSpeechData.py:35, KWSFSL/data/MSWCData.py:35

### Prototype-Based Inference

- Description: Classification is performed by computing the distance between a test embedding and class-mean prototypes.
- Why it appears to exist: Enables on-device customization without retraining the entire network.
- Examples: `KWSFSL/classifiers/NCM.py`
- Evidence: KWSFSL/classifiers/NCM.py:50

### UV-First Execution

- Description: All scripts and dependency management are handled via `uv`.
- Why it appears to exist: Ensures reproducible environments and consistent execution across different machines.
- Examples: `uv run KWSFSL/metric_learning.py`, `uv add <package>`
- Evidence: uv.lock, pyproject.toml

#!/bin/bash
TARGET_DIR=$1
INDEX_FILE="$TARGET_DIR/index.md"

echo "# Index" > "$INDEX_FILE"
echo "" >> "$INDEX_FILE"

for f in "$TARGET_DIR"/*.md; do
    filename=$(basename "$f")
    if [ "$filename" != "index.md" ]; then
        echo "<!-- Imported from: ./$filename -->" >> "$INDEX_FILE"
        cat "$f" >> "$INDEX_FILE"
        echo "" >> "$INDEX_FILE"
        echo "<!-- End of import from: ./$filename -->" >> "$INDEX_FILE"
        echo "" >> "$INDEX_FILE"
    fi
done
